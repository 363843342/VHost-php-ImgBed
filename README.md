# VHost-php-ImgBed
基于虚拟主机网页面板php的图床，包含php服务端和浏览器扩展插件的客户端

# 图床搭建与插件配置教程

* * *
🛠️ 图床搭建教程
----------

### 1\. 环境准备与配置

*   **PHP 版本设置**：在虚拟网页空间主机面板中，将 PHP 版本切换至 **8.x** 。若无 8.x 版本，则选择 **7.x** 。
*   **域名准备**：注册域名并在虚拟主机面板中完成**域名绑定** 。
*   **修改默认密码**：
    1.  编辑打开 `index.php` 文件 。
    2.  找到 `$safe_key = ''` 变量 。
    3.  使用在线 **SHA256** 工具加密你的明文密码，并将生成的密文填入引号内 。

### 2\. 文件上传

*   **上传方式**：通过在线文件管理将 `index.php` 上传至面板 。建议打包压缩包.zip上传
    > **注意**：若因 WAF 或 CDN 防火墙拦截包含代码的可编辑类文件导致 IP 被禁两小时（常见于文本上传），请将文件打包成 `.zip` 或 `.7z` 格式上传后在面板内解压 。
*   **版本适配**：若主机仅支持 PHP 7.x，请上传 `index 7.php` 并手动更名为 `index.php` 。
*   **目录建议**：建议创建一个专门的文件夹放置 `index.php`，因为后续上传的图片会与该文件存放在同一目录下 。

### 3\. 初始化与测试

*   **访问登录**：访问 `域名/index.php`，输入**明文密码**尝试登录 。
*   **容量限制**：登录后可在右上角手动设置图床可用容量（单位：MB） 。
    *   _说明_：该功能仅检测同目录下的图片文件占用，不计算其他文件 。手动限制可防止因空间超额导致的停机或删机 。
*   **功能测试**：测试上传与删除功能 。
    *   _提示_：目前不支持批量删除，如需批量操作请前往主机面板 。

* * *

🔌 浏览器插件（客户端）配置
---------------

**插件名称：** Clipboard Image Uploader

### 4\. 安装插件

*   解压插件包，在浏览器扩展管理中开启**开发者模式**，选择“加载已解压的扩展程序” 。
*   插件图标为一个 **“剪”** 字 。

### 5\. 添加服务器

*   点击图标，进入左下角的**管理服务器** 。
*   添加服务器 URL（必须以 `index.php` 结尾），例如 ：
    *   `http://example.cc/index.php`
    *   `https://example.cc/img/index.php`
*   **更新提示**：若添加已存在的 URL，系统会提示密码已更新 。

### 6\. 备份与管理

*   **导入导出**：备份文件采用 **JSON** 格式 。
*   **冲突处理**：导入备份时，若存在相同链接，插件会保留现有记录而非覆盖更新 。
*   **删除**：点击“删除当前”按钮可移除不需要的服务器链接 。

### 7\. 使用技巧

*   **粘贴图片**：支持 `Ctrl + V` 或右键点击框内粘贴 。
    *   _小贴士_：若 `Ctrl + V` 无效，请点击一下粘贴框外，使鼠标焦点回到插件上 。
*   **文件上传**：左键粘贴框内部可通过文件管理器选择图片 。
*   **获取链接**：上传成功后会自动复制图片 URL，或点击下方的 URL 手动复制 。
*   **容量同步**：插件显示的剩余容量与网页端并非完全实时同步，通常在日期变动或上传新图片时刷新 。
